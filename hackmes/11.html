Cross Site Scripting User Agent
XSS is a vulnerability that allows attacker to inject client-side script into web-pages.
XSS can be persistent or non-persistent.
It is persistent when you inject the code and it's stored by the server.
It is non-persistent when you inject the code and it isn't stored by the server but just executed in the browser.
To break a vulnerable form via XSS you have to (usually) close tags then start your own code.
That type of vulnerability is usually used to steal cookies and gain administrator privileges.
